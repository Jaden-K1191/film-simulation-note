Film Recipe Note v6
===================

배경
----
- 사진 배경: 상단 '배경' 버튼에서 사진 선택
- 단색 배경: 상단 '기본' 버튼에서 색상 선택
- 제공 색상:
  #FFFFFF
  #F5F5F7
  #DCE4DE
  #1D2A3A
  #121212
  #FFECF1
  #FFF9E6
  #040720
- 기본값은 #121212
- 밝은 배경을 선택하면 첫 화면 텍스트가 자동으로 어두운 색으로 전환

레시피 입력 순서
----------------
1. 필름 시뮬레이션
2. 모노크롬 색상
3. 그레인 효과
4. 컬러크롬 효과
5. 컬러크롬 FX 블루
6. 피부 보정 효과
7. 화이트 밸런스
8. 다이내믹 레인지
9. 톤 곡선
10. 색농도
11. 샤프니스
12. 고감도 노이즈 감소
13. 선명도(명료도)
14. 장노출 노이즈 리덕션

추가 변경
---------
- LMO 삭제
- 모든 숫자 화살표 입력을 선택형 메뉴로 변경
- ISO: 125 이상 / 250 이상 / 500 이상
- 노출 보정: -3 ~ +3, 1/3 스탑
- WB Tint: R/B 각각 -9 ~ +9
- 색 온도 선택 시 K값 직접 입력
- Monochrome/ACROS 선택 시 STD, Ye, R, G 필터 선택
- Monochrome/ACROS 선택 시 WC/MG -9 ~ +9 표시
- 다른 시뮬레이션에서는 모노크롬 설정을 #N/A 및 비활성화
- 기존 IndexedDB 레시피와 대표 사진 유지
- 기존 필드 값은 새 구조로 자동 변환

백업
----
백업 파일에는 다음이 포함됩니다.
- 전체 레시피
- 레시피 대표 사진
- 앱 배경 사진 또는 단색 배경 설정


v6.1 변경
---------
- 단색 배경 8종별 전용 텍스트 컬러 적용
- 밝은 배경은 검정 일괄 적용이 아니라 배경색에 맞춘 짙은 중성색 사용
- 어두운 배경은 크림 또는 청회색 계열 사용
- 첫 화면, 상단 제목, 보조 문구, 상단 도구 버튼 색상 자동 전환
- 레시피 이름/카테고리 2분할 배치
- ISO 기준/노출 보정 2분할 배치
- ISO 표시를 ISO 125 이상 / ISO 250 이상 / ISO 500 이상으로 변경
- 컬러크롬 효과/FX 블루/피부 보정 효과 3분할 배치
- 다이내믹 레인지/톤 곡선 2분할 배치
- 색농도/샤프니스/고감도 노이즈 감소/선명도 4분할 배치
- 모든 선택형 메뉴의 선택값 중앙 정렬
- 좁은 화면에서는 항목명 길이에 따라 글자 크기 자동 축소


v6.2 변경
---------
- 레시피 목록 카드에서 카테고리를 더 명확하게 표시
- 레시피 이름 오른쪽에 독립된 카테고리 배지 추가
- 작은 화면에서는 배지 크기 자동 축소
- 레시피를 열지 않아도 카테고리를 즉시 확인 가능


v6.3 폰트 변경
--------------
- 한글 UI, 설명, 버튼, 설정 항목: Spoqa Han Sans Neo
- 영문 제목, 레시피 이름, 카테고리, 숫자 설정값: Avenir Next
- Spoqa Han Sans Neo는 CDN CSS로 불러옴
- Avenir Next는 Apple 기기 내장 폰트를 우선 사용
- Avenir Next가 없는 기기에서는 Helvetica Neue / Helvetica / Arial 순으로 대체
- 기존 레이아웃, IndexedDB 데이터, 대표 사진, 배경 설정 유지
- Service Worker 캐시 이름을 v6-3으로 변경
